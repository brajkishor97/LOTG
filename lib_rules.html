About Library OTG
  • Library OTG provides hassle-free surfing of available books in library
  • Surfer can request any book from anywhere around in campus
  • Book Collection and Return is to be done physically
Get Login details
  • Ask the Library Incharge to make your account
  • You will be given a temporary password
  • Change it as soon as you login first time
  • Keep your password confidential
Surfing Library
  • You can view any book by pressing 'View Book' button
  • Can request any book by pressing 'Book it' button
  • Can see total copies, total request of any book at present
  • Can review any book and also see other reviews
  • Can see search for, filter books
Caution
  • It is advised to Request book cautiously, as the request can only be cancelled by Library Incharge
  • Book cannot be issued if have any other book request
  • Don't submit false/absurd/abusive reviews. It is all public and tied to your account
  • There is 1 reference book of every type, which cannot be issued.
  • If book requests exceed the total no. of books, you can lose book if you show up late in library.
  • Max. Book requests : 2 (When 0 issued books ), : 1 ( When 1 book issued ), and : 0 ( When 2 books issued )
Fines
  • A book can be issued for max. 7 days ( inclusive )
  • Fine of Rs.10/- for every next group of 7 days.
  • Eg: Returned after: (1) 9days: 10/- (2) 14days: 10/- (3) 15days: 20/-
